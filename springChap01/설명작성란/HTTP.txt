HTTP - Hypertext Transfer Protocol
클라이언트와 서버 간의 통신을 위한 프로토콜
주로 웹에서 데이터를 주고 받을 때 사용
메서드, 헤더, 바디로 구성

메서드
-GET : 주로 API나 DB에서 데이터를 가지고 오는데 사용
-POST : 데이터베이스나 서버에 데이터를 제출하고 싶을 때 사용

헤더
인증 토큰이나 캐시, 사용자 정보와 같은 내용은 meta나 데이터 헤더부분을 이용해서 전달
직접적으로 보이지는 않지만 중요한 정보를 담을 때 사용

바디
주로

RESTful
-Resource : url 주소에 담겨있는 모든 정보
-Representation
-Stateless
-Uniform Interface
